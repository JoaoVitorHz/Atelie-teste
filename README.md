# Intro
Landing Page onde você consegue realizar um cadastro ou fazer um login, lembrando que este projeto ainda não esta pronto!

## Tecnologias utilizadas
- React
- Typescript 

## Instalação
`npm install atelie-teste`

## Iniciar a aplicação localmente 
`npm run dev`
